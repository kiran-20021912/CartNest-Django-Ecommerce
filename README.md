# CartNest – Django E-commerce Website

## Overview
CartNest is a full-stack e-commerce web application built using Django and PostgreSQL.

## Tech Stack
- Python
- Django
- PostgreSQL
- HTML
- CSS
- Bootstrap
- JavaScript

## Features (Planned)
- User Authentication
- Product Catalog
- Shopping Cart
- Wishlist
- Checkout
- Payment Integration
- Order Management
- Admin Dashboard
